Lodash

– is a JavaScript library which provides utility functions for common programming tasks using the functional programming paradigm.

A little about Language

_.is functions for determining type (instead of var == null.)
( _.isUndefined, _.isString, _.isNumber, _.isArray )

_.to functions for converting type.
( _.toArray, _.toInteger, _.toNumber, _.toString)

Main areas:

  • 1. Utilities
  • 2. Function
  • 3. String
  • 4. Array
  • 5. Collection
  • 6. Object
  • 7. Seq

Math, Date, Number, and Util:

  • Functions for simplifying dealing with math, date and number operations. (_.floor, _.max, _.min, _.round, _.clamp, and _.now.)
  • _.bindAll function causes all methods belonging to an object to have the same context.

Function

_.bind replaces call(), apply() or bind().

Caching via _.memoize

Currying via _.curry and _.curryRight

Limiting call counts via _.once, _.before, and _.after

String

  • Character change via :
    _.lowerCase, _.toLower, _.upperCase, _.toUpper, _.capitalize, _.camelCase, _.kebabCase, _.upperFirst, _.replace, etc.
  • String conversion via:
    _.pag, _.trim, _.words, _.split, _.repeat, etc.

Arrays

truncation and splitting via:
_.slice, _.compact, _.difference, _.chunk, _.drop, _.intersection, _.pull, _.remove, etc.

combination via:
_.join, _.fill, _.concat, _.union, _.intersection, _.xor, etc.

modification via:
_.reverse, _.flatten, and _.flattenDeep

creation and compression via:
_.fromPairs, _.zip, _.unzip, etc.

Why lodash?

  • 1. Keeps code minimal and neat.

    
								_.intersection([2, 1], [2, 3]);
								// => [2]
								
								
								_.union([2], [1, 2]);
								// => [2, 1]

  • 2. Easier to code. Just to remember some functions

    
								var arr = ['apple', 'banana', 'strawberry', 'banana', 'orange']; 
								
								var arr = _.concat(arr, 2, [3], [[4]]);
								
								console.log(arr);
								
								// output: ["apple", "banana", "strawberry", "banana", "orange", 2, 3, Array(1)]

  • 3. Even newbies can understand

    
								_.uniq([2, 1, 2]);
								// => [2, 1]

More Examples

Loops

Pure JS


							var arr = ['apple', 'banana', 'strawberry', 'banana', 'orange'];
							for(var i = 0; i < arr.length; i++) {
							
							
								console.log('fruit: ' + arr[i]);
							}

Lodash


							var arr = ['apple', 'banana', 'strawberry', 'banana', 'orange'];
							_.each(arr, function(fruit, index, fruits) {
								
								
							console.log('fruit: ' + fruit);
							});

Grouping array by length of string with _.groupBy


							_.groupBy(['apple', 'banana', 'orange'], 'length');
							// => { '0': ['apple'], '1': ['banana', 'orange'] }

Finding the object and extending it: _.find and _.extend.


							var arr1 = [{'fruit': 'apple'}, {'fruit': 'banana'}, {'fruit': 'strawberry'}, {'fruit': 'banana'}, {'fruit': 'orange'}]
							
							
							var arr = _.find(arr1, function(obj){ 
							
														
								if(obj.fruit == "apple"){
							
							

									arr1.push(_.extend(obj, {"quantity": 20}));
								}
							});
							
							
							console.log(arr1)
